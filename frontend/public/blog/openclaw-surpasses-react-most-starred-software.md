Just last week, we wrote about [OpenClaw surpassing Linux](/blog/openclaw-surpasses-linux-14th-most-starred) to claim the #14 spot on GitHub's all-time star leaderboard. At the time, React sat at 243K stars, and we noted that "at this rate, it might not hold that lead for long."

It didn't.

OpenClaw has now crossed 250K+ stars, overtaking React to become the most-starred non-aggregator software project on GitHub — a title React held unchallenged for years. From zero to #1 in under four months, with no sign of slowing down.
